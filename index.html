Firebase — Real-time Multi-User
⚡ โหมด: Local (เครื่องนี้เท่านั้น)

วิธีเชื่อมต่อ Firebase (ทำครั้งเดียว ~10 นาที)

1. ไปที่ console.firebase.google.com → สร้าง Project ใหม่ (ฟรี)

2. เลือก Firestore Database → Create database → Test mode

3. Project Settings → เพิ่ม Web App → Copy firebaseConfig

4. วาง JSON Config ด้านล่าง → กด เชื่อมต่อ Firebase

5. ส่งไฟล์นี้ให้ทีม ทุกคนวาง Config เดิม → ข้อมูล Sync Real-time ✅
